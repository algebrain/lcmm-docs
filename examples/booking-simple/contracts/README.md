# Контракты модулей `booking-simple`

Этот каталог содержит контракты модулей простого примера:

- `booking.contract.yaml`
- `notify.contract.yaml`
- `audit.contract.yaml`

Контракты описывают внешнее поведение модулей:

- HTTP-маршруты;
- события, которые модуль публикует или потребляет;
- роль модуля в общем demo-потоке.

`app.main` и `app.sqlite` здесь не выделены в отдельные module contracts, потому что в этом примере они выполняют роль composition glue и слоя хранения, а не самостоятельных бизнес-модулей.
